ติดตั้ง Website ด้วย Github

เข้า Website Github พร้อมทั้งเข้าสู่ระบบ และคลิก Dropdown จากนั้นเลือก New repository

เมื่อกดเข้ามาแล้ว ให้ตั้งชื่อ Repository name (อะไรก็ได้) และติ๊ก Checkbox ที่ตัวเลือก Add a README file

และทำการกด Create repository

เมื่อเข้ามาหน้านี้ ให้กด Add file และเลือก Upload files

อัพโหลดไฟล์งานของเรา (สามารถลากไฟล์ลงไปได้เลย) ต้องเป็นไฟล์ทุกอย่างที่เกี่ยวข้องกับงานเช่น รูปภาพ

เมื่อเข้ามาหน้านี้ ให้ทำการกด Setting มุมบนขวาของจอ

กดหัวข้อ Pages และตั้งค่า Branch เป็น main และ /(root) และกด Save และรอสักพักจะมีหน้าต่างป๊อปอัพขึ้นมาข้างบน ในป๊อปอัพจะมีลิงค์ Website ของเราอยู่สามารถกดเข้าได้เลย

Thank you for visiting the website.
